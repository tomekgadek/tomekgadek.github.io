Pętla for w języku C

W języku C, pętla for jest jednym z podstawowych narzędzi do iteracji przez zestaw instrukcji. Umożliwia wykonywanie kodu wielokrotnie w określonym zakresie. Poniżej przedstawiamy przykładowy kod oraz kilka ważnych uwag dotyczących użycia pętli for.

#include <stdio.h>

for (int i = 0; i < 100; i++) {
    // Wyświetla liczby od 0 do 99
    printf("%d\n", i);
}

int main() {
    for (int j = 0; j < 5; j++) {
        // Inna pętla for
        printf("Liczba: %d\n", j);
    }

    return 0;
}

W powyższym kodzie pętla for wykonuje się 100 razy, drukując liczby od 0 do 99. Pętla wewnętrzna w funkcji main wykonuje się 5 razy, wyświetlając liczby od 0 do 4. Pętle for mogą być zagnieżdżane i dostosowywane do różnych potrzeb programistycznych.
