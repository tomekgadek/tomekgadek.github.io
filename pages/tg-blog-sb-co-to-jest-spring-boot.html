Co to jest Spring Boot?

Co to jest Spring Boot?

Spring Boot to framework oparty na Spring Framework, który upraszcza tworzenie aplikacji Java. Pozwala programistom szybko uruchamiać projekty bez konieczności ręcznego definiowania konfiguracji.

Kiedyś to było ;)

Dawniej Spring Framework wymagał żmudnej konfiguracji w XML. Należało ręcznie tworzyć komponenty, ustawiać zależności i konfigurować serwery. W 2014 roku pojawił się Spring Boot, który uprościł ten proces dzięki automatycznej konfiguracji.

Założenia

Spring Boot jest to rozwiązanie convention over configuration, które upraszcza proces tworzenia aplikacji.

Opiera się na kilku kluczowych założeniach.

  • Łatwe uruchamianie. Minimalna konfiguracja wymagana do uruchomienia aplikacji.
  • Autokonfiguracja. Spring Boot automatycznie konfiguruje aplikację.
  • Szybkość. Spring Boot przyspiesza proces tworzenia aplikacji dzięki wbudowanym funkcjom i automatyzacji.
  • Wbudowany serwer aplikacji. Może działać jako samodzielna aplikacja z wbudowanym serwerem (np. Tomcat, Jetty).

Spring Boot = Spring Framework + Kontener Aplikacji + Konfiguracja
