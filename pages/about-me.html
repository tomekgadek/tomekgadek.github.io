O mnie

Nazywam się Tomasz Gądek. Od 2014 roku pracuję jako programista Java. Od 2017 roku prowadzę zajęcia na Akademii Tarnowskiej. Jestem absolwentem Uniwersytetu Jagiellońskiego w Krakowie. Łączę pracę zawodową z prowadzeniem zajęć na uczelni, co pozwala mi dzielić się praktyczną wiedzą i doświadczeniem ze studentami.

Prowadzę zajęcia z zakresu:

  • języków znacznikowych,
  • testowania i jakości oprogramowania,
  • narzędzi i środowisk programistycznych,
  • interfejsów użytkownika,
  • technologii informacyjnej.

Oprócz programowania tworzę kursy programistyczne. Szkolenia są skierowane głównie do osób, które nie potrafią programować, ale chcą nauczyć się podstaw.

Kursy IT, które tworzę, obejmują szeroki zakres technologii i języków programowania, takich jak:

  • Java, Kotlin
  • C#, C++, C
  • Angular, React, Node, JavaScript
  • Go
  • Rust
  • PHP
  • Python
  • Haskell

Dodatkowo zajmuję się tworzeniem narzędzi, które wykorzystują API popularnych dużych modeli językowych (automatyzacja pracy).
